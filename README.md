# Zenn CLI

`npx zenn new:article`
`npx zenn new:article --slug testtesttest --title test`

- [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

### 手順

テーマ選び
slug 考えコマンド
prompt ファイル

- title 雑にいれる
- 細かい要素書く

title 修正、全体の文章みる
topic に word いれる
